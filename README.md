### Summary

This repository contains all the resources and materials that I have used to learn the Mathematics for Machine Learning and Data Science Specialization offered by DeepLearningAI on Coursera. The specialization consists of 3 courses:

**1. Linear Algebra for Machine Learning and Data Science**

**2. Calculus for Machine Learning and Data Science**

**3. Probability & Statistics for Machine Learning & Data Science**
